# Certificado da 3ª Imersão DEV da Alura

Neste repositório, há o "CertifiCard" que fizemos na última aula da Imersão Dev organizada pela Alura.

Utilizamos HTML, CSS e JavaScrypt para desenvolver uma página web contendo nossas informações e links paras as redes sociais e projetos desenvolvidos durante a Imersão.

Todos os códigos foram escritos no editor online CodePen.
